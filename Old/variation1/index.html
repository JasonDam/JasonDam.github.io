Variation 1: A static map of the Calgary LRT map which displays all LRT stations for Calgary Transit. The visualization is currently selected at Marlborough Station. Information about crime count in the surrounding communities of the selected station is displayed in a card. This visualization focuses purely on the stations and does not display any geo data of the communities. The LRT station map is also not an accurate representation of how the stations are dispersed geographically. However, this map evenly spaces stations apart from each other making it easy to view every station.
